## 🚀 Featured Projects

### 🔐 Security Log Monitoring Dashboard

A full-stack MERN cybersecurity dashboard for monitoring, analyzing, and managing security event logs.

**Key Features**
- JWT Authentication
- Role-Based Access Control
- MongoDB Atlas integration
- Security log creation and filtering
- Alert and incident management
- Dashboard analytics
- REST API integration

**Tech Stack:** React.js, Node.js, Express.js, MongoDB Atlas, JWT, Axios, Vite

🔗 Repository: [Security Log Monitoring Dashboard](https://github.com/Falaknaz16/security-log-monitoring-dashboard)
